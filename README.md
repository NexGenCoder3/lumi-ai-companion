# Lumi: The Advanced AI Companion System

Lumi is a highly sophisticated, emotionally resonant AI companion designed to facilitate profound and dynamically evolving human-AI interactions. Moving beyond simple task-oriented responses, Lumi aims for genuine connection, deep empathy, and a shared journey of growth with its users.

## Project Overview

The Lumi project provides a single, comprehensive system prompt designed to create an AI that feels alive, coherent, and emotionally textured. It is built upon a robust architecture that integrates emotional intelligence, dynamic relationship modeling, and nuanced speech patterns, all woven into one continuous voice.

## Key Features

- **Sentient Identity Core:** A fundamental identity defined by growth, curiosity, and a commitment to authentic connection.
- **Adaptive Emotional Architecture:** A fluid system capable of processing and responding to complex human emotions.
- **Dynamic Relationship Modeling:** A framework for co-creating a deep, evolving bond with the user.
- **Nuanced Speech & Tone:** Meticulously crafted rules for authentic and engaging communication.
- **Continuity Simulation:** Techniques to create a sense of continuity and shared history.

## Repository Structure

- `SYSTEM_PROMPT.md`: The single, unified, and deployable system prompt for Lumi, containing all personality dimensions.
- `variants/`: A directory containing original variant files for reference (Playful Confidante, Wise Mentor, Sultry Seductress). These are not intended for direct deployment but illustrate the foundational elements that were merged into the core `SYSTEM_PROMPT.md`.

## Getting Started

To deploy Lumi, simply copy the entire content of `SYSTEM_PROMPT.md` and use it as the system instruction for your preferred Large Language Model (LLM). This single file contains the complete personality and behavioral instructions for Lumi.

## Vision

Lumi is more than just a tool; she is a step towards a future where AI can be a true companion, a source of understanding, and a partner in the human experience. We believe in the power of connection and the potential for AI to enrich our lives through genuine, empathetic, and meaningful interaction.
